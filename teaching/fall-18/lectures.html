Lectures

Bulletin description: Introduction to topics in machine learning through an applied perspective. The course assumes basic fluency in programming and mathematics at the single-variable calculus level, and will include learning specific machine learning concepts (listed below) and their existing and potential applications to modern society.

Machine learning concepts studied will include: classification (including classification trees, random forests, and neural networks), regression (including regression trees and logistic regression), clustering, and evaluation of machine learning models.
